Icon Action Description
Select Use to select a node or drag around the canvas.
Magnify Click on the canvas to zoom in or out. You can also scroll to zoom on desktops
Pan Click on the workspace whilst a pan direction is selected to move in that direction
Create Node This will add a new node to the workflow
Create Child Node This will create a child node inside a selected node
Link Selecting two nodes will create an link pointing from the first to the second
Delete Removes a node or an edge from the workflow. Clicking a parent node will remove its children too

    Adding nodes and links

  • Click on in the toolbar to select the Add node tool
  • Create nodes on the graph by clicking on points in the diagram while is 'on'
  • To create a child node, select the Add child node tool and then click on a parent node in the diagram where you want to add a child to
  • Click on in the toolbar to select the Link tool
  • While the tool is 'on', click on the nodes in the graph: the first node you select will be the source of the link, the second will be the target

    Deleting nodes and links

  • Click on in the toolbar to select the Remove node tool
  • Delete nodes or links by selecting them in the diagram

    Zooming and panning

  • Select tool from the toolbar (click and hold to get other panning options to appear) then click anywhere on the diagram to pan in the selected direction
  • Select the Zoom in tool from the toolbar (click and hold to get the Zoom out option to appear), then click anywhere on the diagram to apply the zoom
